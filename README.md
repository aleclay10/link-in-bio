## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Setup

```bash
git clone https://github.com/aleclay10/link-in-bio.git
cd link-in-bio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
