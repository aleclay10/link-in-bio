# CLAUDE.md

## Project Overview

Link-in-Bio page builder using Next.js with App Router, TypeScript, and Tailwind CSS.

## Commands

- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run lint` — Run ESLint
- `npm run start` — Start production server

## Conventions

- Use TypeScript for all files
- Use the Next.js App Router (app/ directory)
- Use Tailwind CSS for styling (no separate CSS files)
- Use functional React components with hooks
- Keep components small and focused
- Use lowercase-kebab-case for file/folder names in app/

## Project Structure

```
app/           — Next.js app router pages and layouts
components/    — Reusable React components
public/        — Static assets
```
